# Factorial vs Exponential

[:link: Puzzle on CodinGame](https://www.codingame.com/training/medium/factorial-vs-exponential)

**Level:** medium
**Topics:** Mathematics

## Goal 

For each of the given numbers A, find the smallest integer N, such that **A^N < N!** , where **N! = 1 \* 2 \* ... \* N**  
The numbers given can have up to 2 digits after decimal point. 

Input

**Line 1:** An integer K for the number of inputs.  
**Line 2:** K space separated numbers (can have a fractional part, e.g. 1.5): A\_1, A\_2, ... , A\_K

Output

**Line 1:** K space separated integers: N\_1, N\_2, ... , N\_K.

Constraints

1 ≤ K ≤ 100  
1 < A\_i < 10000

Example

Input

2
2 3

Output

4 7
