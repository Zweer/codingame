# Great escape

[:link: Puzzle on CodinGame](https://www.codingame.com/multiplayer/bot-programming/great-escape)

**Level:** multi
**Topics:** Pathfinding, Minimax

Write a program that calculates the sum of a series of integers.   

  
**INPUT:** 

**Line 1:** The number N of integers to add 

**Line 2:** The N integers 

  
**OUTPUT:** 

The sum of the N integers. 

**CONSTRAINTS:** 

0 < N < 10000 

**EXAMPLE:** 

**Input** 

3

1 2 3

**Output** 

6
